# 🚀 React Lab: Práticas e Descobertas
> Repositório dedicado a exercícios e pequenas aplicações construídas para solidificar a base do desenvolvimento Front-End com React.

## 🌟 O Propósito da Missão
Como desenvolvedora em formação com foco em **UX/UI**, este espaço serve como meu laboratório para:
1.  **Consolidar a Lógica:** Transformar conceitos teóricos de JavaScript em código React funcional.
2.  **Dominar a Estrutura:** Entender profundamente como os componentes se comunicam e o fluxo de dados.
   
# Dev Quest - Dev em Dobro: Práticas em React

Este repositório reúne todas as práticas realizadas do módulo "React básico" do curso **Dev Quest - Dev em Dobro**. Tudo foi fundamental para aprofundar meus conhecimentos em React, proporcionando experiências práticas e desafios reais.

## Estrutura das Práticas

O repositório está organizado por módulos e exercícios, cada um focado em conceitos essenciais do React:

- **05-estrutura-react**: Estrutura inicial de projetos React com Vite e TypeScript.
- **aula-componentes**: Criação e uso de componentes funcionais e de classe.
- **aula-estilizacao**: Diferentes formas de estilização (CSS global, módulos, styled-components).
- **aula-eventos**: Manipulação de eventos em componentes React.
- **aula-jsx-tsx**: Prática com JSX e TSX, explorando tipagem e sintaxe.
- **aula-listas**: Renderização de listas e uso de keys.
- **aula-props**: Passagem e manipulação de props entre componentes.
- **aula-usestate**: Utilização do hook useState para gerenciamento de estado.
- **exercicios**: Exercícios práticos, como contador com limite personalizado, manipulação de inputs, renderização de props, entre outros.
- **todolist-app**: Projeto completo de uma aplicação de lista de tarefas (Todo List), com contexto, hooks customizados e estilização.
- **react-vite**: Estrutura básica de um projeto React com Vite.

## Aprendizados

Durante o curso, pratiquei:

- Estruturação de projetos React modernos
- Criação de componentes reutilizáveis
- Gerenciamento de estado com hooks
- Manipulação de eventos
